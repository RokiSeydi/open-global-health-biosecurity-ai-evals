## WHAT YOU ARE BUILDING

A Domain 1 evaluation pipeline for cultural contextual validity in global health AI deployment.
Specifically: run one eval case (D1_IT_001) against three frontier models, score each response
against a 6-dimension rubric, produce a comparison results table, and visualise the gap between
scaffolded and unscaffolded prompt performance.

This is built on UK AISI's Inspect Evals framework.
